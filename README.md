# turbo-bun-svelte-trpc-starter

[Turborepo](https://turbo.build/) starter with [SvelteKit](https://kit.svelte.dev/) and
[tRPC](https://trpc.io/) running on [Bun](https://bun.sh/). With live reload and
everything you want.

## Stack

- Bun
- bun:test
- tRPC
- Svelte
- SvelteKit
- Vite
- Vitest
- Playwright
- Turborepo
- pnpm
- pnpm-workspaces
- eslint
- prettier
